# ap-csa-archive

**Browse in the browser:** [https://zhoulinhua0-star.github.io/ap-csa-archive/](https://zhoulinhua0-star.github.io/ap-csa-archive/)

A personal, **concept-first** archive of **AP Computer Science A** practice in the spirit of **free-response questions (FRQs)**. Items are grouped by the **programming idea** they stress (conditionals, loops, class design, collections, 2D structures), not by exam year or source.

## What’s inside

Each **problem** lives in its own folder under one of four units. Typical contents:

| File | Role |
|------|------|
| **`code.md`** | Reference solution or scratch implementation (Java, often with part labels like `(a)` / `(b)`). |
| **`README.md`** | Short **core concept** notes (what the task is really testing) and, where it helps, **time complexity** takeaways. |

You can use this repo to **review patterns** before an exam, **compare approaches**, or **link a study note** to a concrete snippet without hunting through full PDFs.

## How it’s organized

The top-level folders mirror common **AP CSA FRQ “families”** (naming is for navigation, not an official College Board mapping):

### `q1-methods&control-structures`

Methods, `if` / `else`, loops, `Math` helpers, simulation-style logic, and parsing simple rules (e.g. leap year, reservations, random experiments).

*Examples:* `roll-die`, `walk-dog`, `hop-simulation`, `score-guessing`, `room-reservation`, `calendar-leapyear`

### `q2-class-design`

Classes with **fields**, **constructors**, and **instance methods**; encapsulation and **object behavior** (no inheritance focus in these snippets).

*Examples:* `APLine`, `SignedText`, `CalorieCount`, `Scoreboard`, `StepTracker`

### `q3-array-arraylist`

One-dimensional **`array`** and **`ArrayList`** traversals, searching, accumulating results, string/list processing, and “process a sequence” FRQ shapes.

*Examples:* `trail-prob`, `TravelPlan`, `WordPair`, `wordlist`, `html-delimiters`, `average-score`

### `q4-2D-array`

**2D arrays**: row/column indexing, nested loops, grids, placement and querying in two dimensions.

*Examples:* `SeatingChart`, `LightBoard`, `latin`, `grid-Location-prob`
